Política de Privacidade

Última atualização: Janeiro 2025

1. Responsável pelo Tratamento

O responsável pelo tratamento dos seus dados pessoais é o McMinsky, com sede em Azinhaga da Cidade 24A r/c, 1750-063 Lisboa, Portugal. NIPC: 517747120

2. Dados que Recolhemos

Podemos recolher os seguintes tipos de dados pessoais:

  • Dados de identificação: nome, data de nascimento, fotografia
  • Dados de contacto: email, telefone, morada
  • Dados académicos: escola, ano escolar, disciplinas, histórico académico
  • Dados de navegação: cookies, endereço IP, páginas visitadas
  • Dados de pagamento: quando aplicável para faturação

3. Finalidades do Tratamento

Os seus dados são tratados para as seguintes finalidades:

  • Prestação dos serviços de formação e acompanhamento contratados
  • Gestão administrativa e faturação
  • Comunicação sobre os nossos serviços e eventos
  • Melhoria dos nossos serviços e experiência do utilizador
  • Cumprimento de obrigações legais

4. Base Legal

O tratamento dos seus dados baseia-se em:

  • Execução de contrato: para prestação dos serviços contratados
  • Consentimento: para comunicações de marketing e cookies não essenciais
  • Interesse legítimo: para melhoria dos nossos serviços
  • Obrigação legal: para cumprimento de obrigações fiscais e legais

5. Partilha de Dados

Os seus dados não são vendidos a terceiros. Podemos partilhar dados com:

  • Prestadores de serviços essenciais (alojamento web, email)
  • Autoridades competentes, quando legalmente exigido
  • Encarregados de educação, no caso de menores

6. Cookies

O nosso website utiliza cookies para melhorar a experiência de navegação. Utilizamos:

  • Cookies essenciais: necessários para o funcionamento do site
  • Cookies de preferências: para guardar as suas escolhas (ex: idioma)
  • Cookies analíticos: para compreender como o site é utilizado

Pode gerir as suas preferências de cookies através das definições do seu browser.

7. Conservação dos Dados

Os seus dados são conservados apenas pelo período necessário para as finalidades para que foram recolhidos, ou conforme exigido por lei. Após esse período, os dados são eliminados de forma segura.

8. Os Seus Direitos

Ao abrigo do RGPD, tem os seguintes direitos:

  • Acesso: saber que dados temos sobre si
  • Retificação: corrigir dados incorretos
  • Eliminação: solicitar a eliminação dos seus dados
  • Portabilidade: receber os seus dados em formato estruturado
  • Oposição: opor-se ao tratamento em certas circunstâncias
  • Limitação: limitar o tratamento dos seus dados

Para exercer os seus direitos, contacte-nos através do formulário no website ou nas nossas instalações.

9. Segurança

Implementamos medidas técnicas e organizativas adequadas para proteger os seus dados contra acesso não autorizado, perda ou destruição. No entanto, nenhuma transmissão via Internet é 100% segura.

10. Menores

O tratamento de dados de menores de 16 anos requer consentimento do titular das responsabilidades parentais. Garantimos a proteção adequada dos dados de crianças e jovens que frequentam os nossos serviços.

11. Alterações

Esta Política de Privacidade pode ser atualizada periodicamente. Quaisquer alterações serão publicadas nesta página com indicação da data de atualização.

12. Contactos e Reclamações

Para questões sobre privacidade, contacte-nos através do website. Tem também o direito de apresentar reclamação junto da Comissão Nacional de Proteção de Dados (CNPD).
