"Os dados são o novo petróleo. Mas precisas de saber refiná-los."

Em 4 semanas intensivas, vais dominar Python e as ferramentas essenciais de Data Science. Não importa se nunca programaste na vida ou se já tens alguma experiência. O curso adapta-se ao teu nível e leva-te até ao ponto em que consegues analisar dados reais e tomar decisões informadas.

Aprende com quem trabalha na área todos os dias. Projetos reais, datasets verdadeiros, e técnicas que podes aplicar imediatamente no teu trabalho ou negócio.

Programa do Curso

Semana 1: Fundamentos de Python

  • Variáveis, tipos de dados, operadores
  • Estruturas de controlo (if, for, while)
  • Funções e módulos
  • Projeto: Calculadora de análise financeira

Semana 2: Manipulação de Dados com Pandas

  • DataFrames e Series
  • Limpeza e transformação de dados
  • Agregações e groupby
  • Projeto: Análise de dataset real

Semana 3: Visualização de Dados

  • Matplotlib e Seaborn
  • Gráficos para storytelling
  • Dashboards interativos
  • Projeto: Dashboard de métricas de negócio

Semana 4: Machine Learning Introdutório

  • Conceitos fundamentais de ML
  • Regressão e classificação
  • Avaliação de modelos
  • Projeto Final: Modelo preditivo completo
